# 使用

系统分为 `/api` 和 `/server` 入口。

此处为接口的设计方案，可能存在更新不及时等问题，具体以 `postman` 中显示为准。

> 如遇接口设计不合理等，请及时反馈，不要自己拼接研究怎么使用。

接口文档: https://www.postman.com/somebyteorg/emos

### 正式服信息

建议使用官方的 `https://emos.best` 地址

如遇 `cf` `403` 等问题，可以切换为 `https://api.emos.best` 备用地址。

### 测试服信息

- 地址: `https://test.emos.best`
- TOKEN: `11_test-token`

## 快捷登录

访问主站 会有一个 `密钥`

将 `xxx_xxx` `密钥` 添加到后面 即可快捷登录

如 `https://emos.best/#/sign?token=xxx_xxx`
