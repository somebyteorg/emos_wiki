# 萝卜

> 作为 `emos` 的货币体系，对普通用户的唯一作用就是没有作用。

```txt
签到 
  每次 +1到5
  每天的签到顺序
    第3位+3
    第9位-2
    第11位+2 
    第26位-1 
    第59位+2
    第89位-3
    第113位-2
    第199位-2
    第311位+1
    第457位+2
  连续签到一周 +10
  连续签到一月 +30
  连续签到一年 +3000

幸运发言 
  每101的整数位 +20 
  每199的整数位置 -11

视频方面
  上传有求片的视频 +2 + 加速求片萝卜的70%
  新剧前三首发上传分别 +20 +10 +5
  视频有广告、和已有版本重叠、分辨率过低什么的 被删除 -100
  发现视频有广告 发现者+30

  每天按照播放量汇总 10次播放 = 1萝卜
  每周按照上传的文件大小 四舍五入汇总 100G=1萝卜

  上传字幕 +1

反代方面
  每天结算汇总 
    反代视频流 100次 = 1萝卜
    反代接口的 1000次 = 1萝卜

其他方面
  片单 根据作者定价 每天从用户扣萝卜 给作者60%
  兑换码 3000萝卜 一次邀请资格
  固定密码 100萝卜
  转增萝卜 系统扣20%手续费
  在邮箱中收到用户感谢信时 将给被感谢用户根据实际情况赠送-1000 到 1000萝卜
  高清海报 萝卜达到1000后免费使用
  下载 1G = 10萝卜 （待定）
  赞助 1元 = 100萝卜

根据todb标记的上映日期 前后一周内都算新剧
上传老剧时 先给管理说一声 根据实际情况核算萝卜
```
